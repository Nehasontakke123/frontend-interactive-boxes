# Frontend Interactive Boxes

This project is created as part of the Frontend Technical Support assessment.

## Features
- Three interactive product selection boxes
- Smooth expand/collapse animation
- Dynamic size and colour dropdowns
- Active state highlighting
- Most Popular badge for 2-unit option
- Add to Cart functionality with validation
- Clean UI matching the given Figma design

## Technologies Used
- HTML
- CSS
- Vanilla JavaScript

## How to Run
Open `index.html` in any browser.
